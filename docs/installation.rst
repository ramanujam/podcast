============
Installation
============


Stable release
--------------

To install podcasts, run this command in your terminal:

.. code-block:: console

    $ pip install podcasts

This is the preferred method to install podcasts, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for podcasts can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/ramanujam/podcasts

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/ramanujam/podcasts/tarball/master

Once you have a copy of the source, you can install into poetry virtual environment with:

.. code-block:: console

    $ poetry install


.. _Github repo: https://github.com/ramanujam/podcasts
.. _tarball: https://github.com/ramanujam/podcasts/tarball/master
